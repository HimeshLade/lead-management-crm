# Lead Management CRM

A full-stack Lead Management CRM built using React.js, Node.js, Express.js, and MongoDB.

## Features

- Add new leads
- View all leads
- Update lead status
- Delete leads
- Search leads by name, email, or company
- Lead statistics dashboard

## Tech Stack

### Frontend

- React.js
- Axios

### Backend

- Node.js
- Express.js

### Database

- MongoDB Atlas

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm start
```

## API Endpoints

- POST /api/leads
- GET /api/leads
- PUT /api/leads/:id
- DELETE /api/leads/:id
- GET /api/leads/search?q=
